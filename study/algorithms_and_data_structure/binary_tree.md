### What's binary tree?
Is a data structure in which each node has at most two children, which are referred to as the left children and right children

```
            ______7_______
           /              \
        __3__           ___11___
       /     \         /        \
      1       5       9         _13
     / \     / \     / \       /   \
    0   2   4   6   8   10    12    14
```
